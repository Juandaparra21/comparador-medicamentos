The goal is to build a medication price comparison platform that retrieves data directly from pharmacy websites in near real-time.
To achieve this, techniques such as web scraping and API consumption (when available) will be used to extract prices, availability, and product details. This data will be normalized and grouped to identify equivalent medications (same active ingredient) and presented clearly to the user.
